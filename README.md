# AutoGuard Counterfactual Lab

用于OTA异常诊断的**分级反事实回放与贝叶斯实验规划器**。

## 核心创新

- 将根因候选表示为带先验概率的假设；
- 根据预期信息增益自动选择下一项实验；
- 参数级、模块级、对象级、环境级干预采用A/B/C/D可信度分级；
- 参数恢复和模块替换优先；
- 像素或天气生成只能作为辅助证据；
- 无法高可信验证时自动生成实车定向测试任务。

## 快速运行

```bash
pip install -e '.[dev]'
python scripts/run_demo.py
uvicorn app:app --reload
```

Demo会从三个根因和三类实验中选择“恢复行人阈值”，执行配对回放并更新后验概率。

## 证据等级

- **A**：参数级单变量回放；
- **B**：模块替换；
- **C**：结构化对象干预；
- **D**：环境/像素生成，不能单独确认根因。

该仓库强调Sim2Real边界，不把生成式场景修改包装成现实因果证明。详见 [SOURCES.md](SOURCES.md)。
